Sportarr Data Directory
=======================

This folder contains your Sportarr configuration, database, and logs.

Contents:
- sportarr.db      - SQLite database with your settings, events, and history
- config.json      - Application configuration (if custom settings are used)
- logs/            - Application log files

IMPORTANT: Back up this folder before uninstalling if you want to preserve your settings!

To migrate to a new installation:
1. Copy this entire 'data' folder to the new installation location
2. Start Sportarr - it will detect and use your existing configuration

For more help, visit: https://sportarr.net
